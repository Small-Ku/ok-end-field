# 自动大世界收菜

## 概述

在大世界（探索地图）中移动时，程序会自动识别可拾取的资源并按 F 键收集，无需手动操作。适合在武陵、四号谷地等区域路过时顺手收取植物、昆虫等材料。

该功能由触发任务「**自动拾取**」提供。触发任务会在后台持续运行，只要玩家身边出现可采集物品，便会自动拾取。

---

## 启用方式

在软件的「**触发任务**」列表中，开启「**自动拾取**」即可。

> 注意：触发任务会在后台持续运行，不会影响其他正在执行的任务（如日常任务、自动送货等）。

---

## 自动拾取的物品

程序内置**白名单**，仅拾取以下物品：

| 类型 | 物品名称 |
| ---- | -------- |
| 植物 | 荞花、灰芦麦、苦叶椒、柱状菌、酮化灌木、多齿叶、砂叶、芽针 |
| 果实 | 柑实 |
| 昆虫 | 萤壳虫、灼壳虫 |
| 交互 | 采集、打开、触碰、激活（通用交互提示） |

---

## 不会自动拾取的物品（黑名单）

以下物品被列入**黑名单**，程序不会自动拾取：

- 协议核心
- 激活箱子

---

## 武陵区域主要收集物

武陵地区的大世界收菜以以下材料为主，用于合成武陵特色产品：

| 原料 | 用途 |
| ---- | ---- |
| 芽针 | 合成「芽针针剂」「优质芽针针剂」 |
| 多齿叶 | 合成「锦草软饮」「优质锦草软饮」 |

> 这些产品可用于据点兑换调度券，详见 [日常任务.md](../docs/日常任务.md#据点兑换) 。

---

## 工作原理

1. 程序检测当前是否处于大世界（非战斗/副本界面）。
2. 识别屏幕上是否出现「按 F 交互」按钮。
3. 读取按钮旁的文字标签：
   - 若文字在**黑名单**中，跳过。
   - 若文字在**白名单**中，直接按 F 拾取。
   - 若文字不在名单中，通过图标颜色辅助判断：若图标不是灰色/白色（即高亮可拾取状态），则自动拾取。
4. 拾取后等待 0.2 秒，继续检测下一个可拾取物品。

---

## 注意事项

- 该功能为**被动触发**，不会主动寻路前往采集点，需要玩家手动或配合其他任务在地图上移动。
- 若不希望自动拾取某类物品，可在源码 `src/tasks/AutoPickTask.py` 中修改 `white_list` 或 `black_list`。
- 触发任务在后台运行时，若游戏画面切换到非大世界状态（如进入副本），程序会自动暂停拾取。
